# 🎓 Student Flow Manager - Queue-Based Student Management System

**🌟 Streamline student records with efficient Queue Data Structure operations**

---

## ✨ Features

- ⚡ Instant Enrollment – Add students in real-time with a single click  
- 🔍 Quick Search – Find any student record in seconds  
- ✏️ Easy Updates – Modify student details effortlessly  
- 🔁 Queue-Based Processing – FIFO order using Linked List implementation  
- 🎛️ Simple GUI – User-friendly Swing dashboard  

---

## 🛠️ Tech Stack

| Category | Technologies Used |
|----------|------------------|
| 💻 Language | Java |
| 🎨 GUI Framework | Swing |
| 🔧 IDE | NetBeans |
| 🏗️ Data Structure | Queue (Linked List) |
| 💾 Storage | In-Memory |
